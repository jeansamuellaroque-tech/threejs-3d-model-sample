# 🚀 Millennium Falcon 3D Viewer

Visualizador 3D del **Millennium Falcon** desarrollado con **Three.js**.
Incluye entorno espacial, iluminación dinámica, piso tipo glass futurista y controles interactivos.

---

## 📸 Vista previa

Proyecto renderizado en navegador con modelo 3D, fondo espacial y UI moderna con Bootstrap.

---

## 🛠️ Tecnologías utilizadas

* Three.js (renderizado 3D)
* GLTFLoader (carga de modelos)
* OrbitControls (interacción de cámara)
* Bootstrap 5 (interfaz)
* HTML5, CSS3, JavaScript (ES Modules)
